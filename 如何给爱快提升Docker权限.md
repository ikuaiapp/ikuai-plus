# 如何给爱快提升Docker权限

！！！如果你本地已经安装好爱快、不方便直接下载定制镜像！！！

爱快由于强大的分流功能，很受大家的喜爱，提供丰富的 VPN 客户端，但提供的 Docker 功能操作便利性实在有限，我这边提供给爱快提升Docker权限、这样通过第三方网页图形化管理容器的复制、迁移、变量调整：）

# 1 托管或绑定爱快云

## 1.1  如何你爱快没有绑定爱快云

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/WgZOZA8QxEPZdqLX/img/430f64a5-aca0-4fe6-81a1-9229477f6ae1.png)

## 1.2  如何已经绑定爱快云

登录爱快云后台，选择网络管理

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/WgZOZA8QxEPZdqLX/img/1077f2a0-d6f4-4a31-bfc9-6050e282b004.png)

选择你的设备，点击托管，弹出对话框，输入我提供的绑定码

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/WgZOZA8QxEPZdqLX/img/4bd0c13a-2694-4d13-ae98-2d25f8d85e4c.png)

# 2 绑定后告诉 等待 我插件推送

## 2.1 推送需要爱快登录用户名和密码 ，需要提供

## 2.2 插件生效、需要重启、请在网络不繁忙联系我操作

# 3 提升后、效果如下

## 3.1 Docker 插件 能用 host 网络,使用Portainer的WEB界面管理容器
